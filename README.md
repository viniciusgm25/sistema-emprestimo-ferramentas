# Sistema de Empréstimo de Ferramentas

Projeto acadêmico desenvolvido com o objetivo de controlar o empréstimo e devolução de ferramentas, aplicando conceitos de lógica de programação e organização de sistemas.

## Objetivo
Facilitar o controle de ferramentas emprestadas, evitando perdas e melhorando a organização.

## Funcionalidades previstas
- Cadastro de ferramentas
- Cadastro de clientes
- Registro de empréstimos
- Controle de devoluções
- Gerador de contrato
- Relatorios
- Registro de multa

## Tecnologias
- Java
- SQL
- Git

## Status do projeto
Em desenvolvimento.
